Layout Modes
Standard Mode (1 Column): Renders a single vertical list of courses.
Magazine Mode (2 Columns): Used explicitly for ELECTIVES due to the high volume of courses. It splits the array of courses in half and renders them side-by-side using display: 'grid', gridTemplateColumns: '1fr 1fr', gap: '1rem'.

5. Print Mode Optimization (@media print)
A massive part of the functionality requires the entire masonry structure to condense onto a single portrait page.

Key Print CSS Rules
Font Scaling: body { font-size: 8px !important; } - drastically shrink everything to prevent text wrapping.
Component Pruning: button, .btn, header, .print-hide { display: none !important; } - Hide global navs, save buttons, and print buttons.
Glassmorphism Stripping: .glass-panel { box-shadow: none !important; backdrop-filter: none !important; padding: 0.15rem !important; } - Remove heavy graphics to save printer ink and space.
Column Squeezing: .course-grid { grid-template-columns: 1fr 22px 22px 22px 22px !important; } - Crush the checkbox columns to exactly 22px to maximize horizontal space for the course names.
Page Break Avoidance: tr, .card { page-break-inside: avoid; } - Prevents category boxes from snapping in half across pages.
